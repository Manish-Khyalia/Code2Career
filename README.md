Code2Career
📌 Overview

Code2Career is a full-stack placement preparation platform where students can manage:

Job Applications
Coding Problems
Resume Management
Dashboard Analytics
AI Interview Preparation

The project is built using Spring Boot, React, MySQL, JWT Authentication, Railway, and Vercel.
